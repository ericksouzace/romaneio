# Romaneio Pijama de Rica — PWA

App/PWA para preencher, salvar e imprimir romaneios da empresa Pijama de Rica.

## Como usar localmente

1. Extraia o ZIP.
2. Abra a pasta `romaneio-pijama-de-rica-pwa`.
3. Dê dois cliques em `index.html`.

## Para instalar como app/PWA

Para o Chrome mostrar o botão de instalação corretamente, o ideal é publicar a pasta em um serviço como:
- Netlify
- Vercel
- GitHub Pages

Depois acesse o link pelo Chrome e clique em “Instalar app”.

## Funções

- Logo da Pijama de Rica no canto esquerdo do romaneio.
- Campos de cliente, telefone, forma de envio, datas e observações.
- Tabela de itens com quantidade, produto, tamanho, cor/estampa, cor/viés e valores.
- Cálculo automático de subtotal, desconto e total.
- Salvar rascunho no navegador.
- Carregar rascunho salvo.
- Salvar arquivo HTML.
- Imprimir ou salvar como PDF.
